## 📋 Descripción
Proyecto iniciado para la cátedra de **Algoritmos y Estructuras de Datos II**. La meta es mejorar la página de manera que sirva como **portfolio personal**.

---

## 🛠️ Stack Tecnológico
Basado en el análisis del prototipo inicial (`index.html`), el sitio utiliza:
* **HTML5**: Estructura semántica básica.
* **Bootstrap 5.1.3**: Framework de CSS para diseño responsivo y componentes.
* **Popper.js**: Para el manejo de tooltips y posicionamiento de elementos (vía Bundle).
